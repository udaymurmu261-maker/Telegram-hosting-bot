# Telegram Hosting Bot

This bot lets you upload and run `.py` or `.zip` files via Telegram.

## Setup Instructions

1. Edit `main.py` and replace `YOUR_BOT_TOKEN` with your own token from @BotFather.
2. Run locally or deploy to Render.com.

### Deploy on Render

- **Build Command:**
  ```bash
  pip install -r requirements.txt
  ```
- **Start Command:**
  ```bash
  python main.py
  ```
